================================================================================
                          QUICK START GUIDE
================================================================================

DEVELOPMENT (local):
  1. ./start-dev.sh
  2. dotnet watch run
  3. Open: http://localhost:5000/swagger

PRODUCTION (Docker):
  1. docker build -t cardduel-api:latest .
  2. Set env vars (JWT_KEY, DB_PASSWORD, etc)
  3. docker-compose -f docker-compose.prod.yml up -d
  4. Visit: https://api.example.com/api/health

KUBERNETES:
  1. kubectl apply -f deployment.yaml
  2. kubectl apply -f service.yaml
  3. kubectl get svc cardduel-api-svc
